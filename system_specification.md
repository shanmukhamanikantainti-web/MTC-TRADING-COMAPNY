# SYSTEM SPECIFICATION DOCUMENT

## Manikanta Trading Company -- E-Commerce Website

------------------------------------------------------------------------

## 1. System Overview

The Manikanta Trading Company E-Commerce System is a web-based
application designed to enable online purchasing of grocery and
wholesale products. It supports both retail customers (B2C) and
wholesale dealers (B2B).

------------------------------------------------------------------------

## 2. System Objectives

-   Provide online grocery and wholesale platform\
-   Support B2C and B2B users\
-   Ensure secure payments\
-   Simplify inventory and order management

------------------------------------------------------------------------

## 3. System Architecture

-   Presentation Layer: Vite + React\
-   Application Layer: Node.js\
-   Data Layer: Supabase

------------------------------------------------------------------------

## 4. Functional Specifications

-   User login & registration\
-   Product browsing\
-   Cart & checkout\
-   Payment via Stripe\
-   Admin dashboard

------------------------------------------------------------------------

## 5. Non-Functional Specifications

-   Performance: \<3s load time\
-   Security: HTTPS, Auth\
-   Scalability: Cloud-based

------------------------------------------------------------------------

## 6. Database

-   Users\
-   Products\
-   Orders\
-   Cart
