# UI CHANGE DOCUMENT

## Manikanta Trading Company -- UI Redesign Prompt

------------------------------------------------------------------------

## Goal

Transform the existing e-commerce UI into a premium, motion-driven,
ingredient-focused design centered around tamarind, red chillies,
cowpeas, and horse gram.

------------------------------------------------------------------------

## Visual Design

-   Use deep red (#8B0000), earthy brown, beige tones\
-   Replace generic images with ingredient-focused visuals\
-   Add natural textures (spices, grains)

------------------------------------------------------------------------

## Motion Design

-   Floating ingredient animations\
-   Scroll-based effects (fade, slide, parallax)\
-   Hover animations (lift + glow)\
-   Add-to-cart animation

------------------------------------------------------------------------

## Homepage

-   Animated hero section\
-   Category cards (Tamarind, Chillies, Pulses)\
-   Smooth product carousel

------------------------------------------------------------------------

## Product Page

-   Zoom images\
-   Bulk pricing animation\
-   Smooth quantity selector

------------------------------------------------------------------------

## UX Improvements

-   Retail / Wholesale toggle\
-   Animated checkout flow

------------------------------------------------------------------------

## Final Style

Premium + Organic + Motion-rich UI inspired by Indian agricultural
products.
